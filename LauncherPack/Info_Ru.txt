1). Создание MiniUpd:
	Для создания мини обновления переносим в папку "Configs&Etc\Parsers\MiniUpd\" все файлы\папки которые нам нужно обновить в клиенте, и запускаем 
	приложение "Parcer_MiniUpd.exe" и нажимаем на кнопку "Старт", он сгенерирует файлик "update.info". Далее берем этот файлик, и файлы 
	обновления и переносим на Web host в директорию домена лаунчера. Все, процесс создания списка обновления завершен.
2). Конфигурация лаунчера:
	Переходит в папку "Configs&Etc\ConfigCreator\" и запускаем приложение "ConfigCreator.exe". 
	Тут я думаю все понятно и без объяснения, но все-же объясню на всякий случай.
	RSS Link - путь к вашей RSS ленте.
	Server IP - IP адрес где установлен ваш сервер. (Для статистики online\offline)
	GS Port - Порт GameServer-a. (Для статистики online\offline)
	CS Port - Порт ConnectServer-a. (Для статистики online\offline)
	Start File - Имя файла, который лаунчер будет запускать. По умолчанию "Main.exe"
	Time Zone - Временная зона.
